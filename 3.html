document.addEventListener()

The document.addEventListener() method is used to add an event handler to the document. We will use this method to create an event handler in our JavaScript file to execute the code inside the method when the document has finished loading

The document.addEventListener() method takes the form:
    document.addEventListener (event, fuction, useCapture)

Parameters:

  • event - string that specifies the name of the event
    For this example, we will use the DOMContentLoaded event to execute the code within the method when the content has finished loading.
    You can find a full list of events here.
  • function - This is a required parameter. It runs when the event occurs
    For this example, this function will contain the code that synchronizes the text to the media
    We will discuss this more next
  • useCapture - Boolean value, indicates whether the event should be executed in the capturing(true) or bubbling (default, false) phase
    This parameter will be false for this example

If you are interested in knowing more about this method go here.

Footnote Plugin

Footnote is a plugin for popcorn that adds a text element to the page. This plugin will be used to add text to the page and synchronize this added text with the media element

The footnote plugin takes the form:
    [nameOfPopcornInstance].footnote({list of options})

Options

  • Start: number, time you want the text to appear
  • End: number, time you want the text to disappear
  • Text: string, text you want to appeat in the target
  • Target: string, id of the element that the text is going to be attached to

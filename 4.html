Section 4: Synching Images with Media

Image Plugin

Now that you have used Popcorn to synchronize text with a video it will be much easier to learn how to use other plugins with Popcorn.

One useful plugin to learn how to use is the image plugin, which is used to synchronize images with media elements.

The format to this plug in is similar to the footnote plugin. It looks like this:

    [nameOfPopcornInstance].image({list of options})

Many of the options for this plugin are also similar to the footnote plugin

Options

  • Start: number, time you want the image to appear
  • End: number, time you want the image to disappear
  • href - string, optional url of the destination of a link
  • Target: string, id of the element that the image is going to be attached to
  • Text: string, text you want to appeat in the target

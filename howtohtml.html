Introduction to the Guide

Welcome to a beginner's guide to Popcorn.js!

To start off, you'll first need to understand what Popcorn.js is. Popcorn.js is a Javascript library which aids the user in creating interactive media, by allowing media elements (video, audio, etc.) to both controland be controlled by other elements of a webpage


Some examples of how popcorn.js can be used are:

  • To synchronize text to an HTML5 video or audio element
  • To synchronize text to an Youtube or Vimeo videos
  • To control a Google map's map with a media element
  • To link images to media elements
  • To link aspects from social media sites such as twitter and facebook to media elements
  • To link news feeds to media elements


The uses for Popcorn are extremely wide-ranging, so the list could go on and on!

The best way to get a better idea of what Popcorn can do is to check out and some examples. A demo that gives a general overview of the Popcorn's many uses can be found here. Other demos, which provide more specific uses of Popcorn, can be found here.

Hopefully, you now have a good idea of what Popcorn is and understand some of it's many uses. As you can see, Popcorn can add value to media and web content and collaboration between media and the web.Popcorn is therefore an extremely useful tool to know how to use for all web dvelopers and film makers. This guide will get you started in learning the basics and start you a journey toward creating interactive media.


This guide will cover:

  • Getting Started
    • Using the Popcorn.js library
    • Adding a video
  • Creating a Popcorn Instance
